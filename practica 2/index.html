- Introducción -

Visual Studio es un entorno de desarrollo integrado (IDE) creado por Microsoft, utilizado para desarrollar software y aplicaciones en una amplia variedad de lenguajes de programación y plataformas. Es una de las herramientas más completas y poderosas para la programación, que facilita la escritura, depuración y mantenimiento de código.

En este programa podremos utilizar distintos tipos de lenguajes de programación como; C++, Python, Java, entre otros.

- Desarrollo -

Para poder crear una pagina web en visual studio, debemos de crear una carpeta nueva en la cual iran distintas sub carpetas las cuales derena tener todo lo que quieras que tenga tu pagina web, videos, imagenes, etc.

Una vez ya tengamos uestra carpeta lista ya podremos abrir nuestra carpeta en el programa y asi poder ir creando la pagina web

Construcción

Ya que tenemos la carpeta nueva lista, entraremos a nuestro programa de visial studio y una vez dentro, dremos clic en File y se abrira una lista con las distintas opiones que puedes realizar ahí, daremos clic en la de new file y nombramos nuestro proyecto como queramos y una ez creado, tendremos una barra a la izquierda de la pantalla, ahí abriremos nuestra nueva carpeta y ahora si empezaremos a programar nuestra pagina web en ella.
