Arduino Electrónica

- Resumen -

En Tinkercad se encontraban los seis circuitos que se iban a realizar con Arduino y servomotores, en los que también se encontraban otros componentes como jumpers, resistencias y luces led. Cada circuito tenía un código diferente en el que se podía ver cómo cambiaban los resultados en los servomotores al girar automáticamente o moverlos por medio de componentes, esto fue para poder diferenciar cada parte de los códigos y cómo cambian los comandos que se usaron y también que componentes se debían usar mientras se cambiaba el código.

- Introducción -

En esa práctica se realizaron 6 circuitos diferentes usando Arduino y servomotores principalmente, esto es para poder ver cómo funciona o se mueve un servomotor dependiendo de cada código y así poder analizar cada componente y al final poder tener una idea más clara.

-ServoMotor

- Desarrollo circuito 1

Conectar todos los componenetes adecuadamente.

Conectamos el servo motor al protoboar, el GND del servo ira conectada al negativo de la protoboard, el VCC ira conectado al positivo del protoboard y la señal ira conectado al pin9 del arduino.

Por ultimo conectamos el negativo de la protoboard al GND del arduino y el positivo d la protoboard al pin de 5V del arduino.

Abrimos arduino y creamos este codigo:

// C++ code //

#include(Servo.h)

Servo servoMotor;

void setup()

{

servoMotor.attach(9);

}

void loop()

{

servoMotor.write(90);

}
